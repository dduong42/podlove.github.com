---
title: Shownotes Entry
---

Shownotes entries are available on episodes via `episode.shownotes`.

###	Title
> entry.title

###	Description
> entry.description

###	URL
> entry.url

Canonical URL.

Defaults to `originalUrl` if no canonical URL is available.

### Has Affiliate Url?
> entry.hasAffiliateUrl

### Original Url
> entry.originalUrl

User provided URL.


### Site name
> entry.siteName

Website name

### Site URL
> entry.siteUrl

Example: The site url of https://example.com/page?param=42 is https://example.com.

### Icon
> entry.icon

Icon URL

### Image
> entry.image

Open Graph or Twitter image.

### Type Icon
> entry.typeIcon

SVG Icon for entry type.

### Type
> entry.type

Either "text" or "link".
